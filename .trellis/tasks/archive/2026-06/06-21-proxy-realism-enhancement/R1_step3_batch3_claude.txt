## 改造策略

### 步骤1：保留 claude.DefaultHeaders 作为兜底
在 `backend/internal/pkg/claude/constants.go`：

```go
// DefaultHeaders 保留为内置兜底常量（不删除，registry 初始化需要它）
var DefaultHeaders = map[string]string{
	"User-Agent": "claude-code/" + CLICurrentVersion + " ...",
	// ... 其他字段
}

// 新增函数：从 registry 读取 claude headers（带 clone）
func GetHeaders(registry *clientidentity.Registry) map[string]string {
	if registry == nil {
		return cloneHeaders(DefaultHeaders)
	}

	claudeSnapshot := registry.Get().Claude
	if len(claudeSnapshot.Headers) == 0 {
		// 兜底
		return cloneHeaders(DefaultHeaders)
	}

	return cloneHeaders(claudeSnapshot.Headers)
}

func cloneHeaders(m map[string]string) map[string]string {
	result := make(map[string]string, len(m))
	for k, v := range m {
		result[k] = v
	}
	return result
}
```
